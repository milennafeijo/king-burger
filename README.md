# Projeto King Burger 

Projeto de uma página de hambúrguer como parte do curso **Dev Quest**, com apoio de ferramentas de IA.

## Tecnologias utilizadas:
• HTML5 <br>
• CSS3 <br>
• JavaScript <br>
• MCP do Figma <br>

## Desenvolvimento

O projeto passou por diferentes etapas:

1. Análise do design no Figma;<br>
2. Estruturação da página utilizando HTML semântico;<br>
3. Criação dos estilos seguindo o guia visual do projeto;<br>
4. Implementação das interações com JavaScript;<br>
5. Revisão e ajustes de detalhes de layout e responsividade.

Esse processo ajudou a desenvolver a habilidade de transformar uma interface planejada no Figma em uma aplicação real, aproximando o desenvolvimento do fluxo utilizado em projetos profissionais.

## Aprendizados 

Ao longo do desenvolvimento, utilizei IA (GitHub Copilot) como uma ferramenta de apoio para esclarecer dúvidas, revisar implementações e identificar pequenos detalhes que passavam despercebidos. 

Também explorei a integração com o **MCP do Figma**, que facilitou a interpretação do layout e ajudou a transformar o design em código com maior fidelidade. Isso tornou o processo de desenvolvimento mais próximo de um fluxo utilizado em projetos reais, onde o desenvolvedor recebe um design pronto e precisa implementá-lo respeitando espaçamentos, tipografia, cores e componentes.

Outro ponto importante foi aprender a revisar o próprio código. Durante o desenvolvimento, várias "pontas soltas" apareceram, como pequenos ajustes de layout, organização do CSS, detalhes de responsividade e melhorias na estrutura do projeto.
